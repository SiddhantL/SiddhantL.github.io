Built to replace the chaos of last-minute event ticketing. High-speed QR scanning, ticket verification, real-time cancellations, and payment processing through RazorPay. Handled 2,000+ transactions in a single night at a university event with zero downtime.

Backend on AWS (EC2 + Cognito for auth), with modular APIs for payment verification, ticket generation, and analytics. Firebase handled real-time attendance updates and duplicate scan detection. Event organizers could set up a fully configured event in under 5 minutes.

Ended up being adopted by 10+ student clubs at my university as the default ticketing system for small to mid-scale events.
